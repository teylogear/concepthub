Variables

Las variables son pequeños contenedores que nos permiten almacenar información temporalmente y después poder trabajar con esa información para hacer lo que necesitemos. Las variables pueden cambiar su valor durante la ejecución de un programa.

Para nombrar las variables utilizaremos identificadores que nos sirven para llamar a la variable cuando queremos trabajar con ella. Un identificador es el nombre que le queremos poner a una variable y puede ser el que sea, pero es preferible que sea descriptivo de lo que contiene.

Existen tipos de variables de acuerdo a su tipo de dato, y pueden ser:

Enteros: 5 -45 2541

Los enteros son números que pueden estar precedidos del signo + o -, y que no tie¬nen parte decimal.


Reales: 34.31 0.8776 -76.6542

Los reales son números que pueden estar precedidos del signo + o -, y que tienen una parte decimal.


Ca¬racteres: ‘a’ ‘$’ ‘-’ ‘?’

Un dato tipo caracter contiene un solo caracter, y se escribe entre apostrofes, aunque este tipo de datos pueden contener números, no pueden ser utilizados para realizar operaciones aritméticas.


Cadena de caracateres: “carlos” “edificio” “ 345-54-12”

Un dato tipo cadena de caracteres contiene un conjunto de caracteres, y se es¬cribe entre comillas.


Booleanos: true false

Un dato booleano solo puede ser verdadero (true) o falso (false)


Ejemplos de variables:

Numero = 3

Nombre = “carlos”

Peso = 58.3

Empleado = true

Esta es solo una forma de visualizar la declaracion de variables, pero es parecida a la de casi todos los lenguajes de programacion.

Constantes

Las constantes son datos que no cambian durante la ejecución de un programa.


Al igual que las variables para nombrar las constantes utilizamos identificadores. Existen tipos de constantes como tipos de datos, por lo tanto, puede haber constantes de tipo entero, real, caracter, cadena de caracteres, etc. Es muy importante que los nombres de las constantes igual que las variables sean representativas de la función que tienen las mismas en el programa.


Dependiendo del lenguaje de programación, la declaración de variables puede variar bastante, pero en todos tienen el mismo funcionamiento.

Operadores aritmeticos

Son los que nos permiten realizar operaciones en nuestros programas, por ejemplo:


+ suma

- resta

* multiplicación

/ divición


Ejemplo de operaciones con estos operadores:


Numero1 = 10

Numero2 = 5

Resultado = numero1 + numero2


Ahora la variable resultado almacena un 15 ya que 10 + 5 =15.

El orden en el que ponemos nuestras variables es importante, esto se representa mejor en la resta


Numero1 = 10

Numero2 = 5

Resultado = numero1 - numero2


Ahora la variable resultado almacena un 5 ya que 10 - 5 = 5.

Pero en el caso inverso

Numero1 = 10

Numero2 = 5

Resultado = numero2 - numero1


Ahora la variable resultado almacena un -5 ya que 5 - 10 = -5.


Para la suma y la multiplicaion el orden no es importante pero para la resta y la divicion debemos tener cuidado con el orden.


Solo podemos hacer operaciones aritmeticas con variables de tipo entero y reales.

Operadores asignación

Sirven para dar valor a las variables


= igual

+= mas igual

-= menos igual

*= multiplicación igual

/= división igual


El signo = es muy simple, lo usamos para iniciar una variable con el valor que deseemos, por ejemplo:


Variable = “valor”

Variable = false

Variable = 13.3


El signo += significa que agregaremos un valor al que ya posee nuestra variable de esta forma


numero1 = 3

numero2 = 6

numero1 += numero2

Es lo mismo que

numero1 = 3

numero2 = 6

numero1 = numero1 + numero2


Los demás signos funcionan de manera similar haciendo las funciones correspondientes de resta, multiplicación y división


Cabe señalar que no debe haber espacios entre los signos. += funcionará pero + = no funcionará
